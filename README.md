## 12306 车票助手

TicketHelper.

基于`PHP`开发一个12306车票助手程序.

---

### Feature

- 余票查询 
- 验证登录（开发中）
- 订单查询（开发中）
- 账号客户信息获取（开发中）
- 配置抢票任务（规划中）

### Installation

环境要求

- PHP7
- Composer

依赖包安装

```shell
composer install
```

### Usage

```shell
php bin/console

输出
Available commands:
  help                Displays help for a command
  list                Lists commands
 ticket
  ticket:quick-query  查询12306列车余票信息.
```
